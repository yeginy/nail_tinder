NAIL DESIGN IMAGES

This directory contains 20 nail design images (nail1.jpg through nail20.jpg) that were downloaded from Unsplash.

All images are free to use under the Unsplash license (https://unsplash.com/license).

If you want to use your own images instead:
1. Replace the existing files with your own images using the same filenames
2. OR update the 'nailDesigns' array in script.js to match your filenames

For best results, use images that are:
- In JPG or PNG format
- Landscape or square orientation
- At least 600px wide
- Clear, high-quality images of nail designs

To download a fresh set of images, you can run the Python script:
```
python3 download_images.py
```
